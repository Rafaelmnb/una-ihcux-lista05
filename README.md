ConversorExpert 

Conversor de moedas desenvolvido para a disciplina de IHC/UX 
Heurísticas Aplicadas

*1. Visibilidade do Status do Sistema**
O resultado da conversão é exibido imediatamente após o usuário inserir o valor, mostrando a moeda de origem, destino e o valor convertido.

*2. Prevenção de Erros**
O sistema usa `try-catch` para capturar entradas inválidas. Se o usuário digitar letras no lugar de números, uma mensagem clara orienta como corrigir.

*3. Estética e Design Minimalista**
A interface exibe apenas o necessário, sem informações desnecessárias, deixando o uso simples e direto.
